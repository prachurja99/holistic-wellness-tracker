# 🌿 Holistic Wellness Tracker

Welcome to **Holistic Wellness Tracker** — a modern, all-in-one app for habit tracking, mood monitoring, journaling, goal setting, and more. Designed for a beautiful and calming experience, this platform supports multiple themes and empowering features to help you thrive in every aspect of wellness.

---

## ✨ Features

- **Unified Dashboard:** Real-time overview of all your habits, moods, journals, goals, and wellness logs.
- **Motivational Quotes:** Drag and drop daily inspiration anywhere on your dashboard.
- **Beautiful Themes:** Instantly switch between Light, Dark, Sepia, Cool Blue, and Vivid color schemes.
- **Responsive UX:** Seamlessly works on desktop and mobile.
- **Modular Trackers:** Habit, Mood, Journal, and Goals sections to stay organized and focused.
- **Smooth & Accessible UI:** Animations, elegant design, and accessible features for stress-free wellness tracking.

---

## 🚀 Technology

- **React** (frontend UI)
- **CSS Variables** for dynamic theming
- **React Draggable** for interactive quote movement
- **Responsive & Modern Design Principles**

---

## 🛠 Installation

Clone the repository
git clone https://github.com/yourusername/holistic-wellness-tracker.git

Navigate into the project directory
cd holistic-wellness-tracker

Install dependencies
npm install

Start the development server
npm start


App runs at [http://localhost:3000](http://localhost:3000)

---

## 🎯 Usage

1. Log new habits, moods, journal entries, and goals.
2. View insights and progress on your dashboard.
3. Drag motivational quotes to keep inspired every day.
4. Change theme to your preferred color mood.
